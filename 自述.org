** 使用 basic usage 

*** 如何安装
本流程使用python与R写成.所以安装本流程之前需要确保系统上Python 与R以及安装完毕. 

**** 安装之前
python 与R的要求. 

流程需要Python3环境. 建议使用anaconda建立的Python 环境.

R 语言环境的建立可以从cran 站点获得. 

**** 简单得安装

如果操作系统中已经安装了git, 那么可以直接在命令行中使用如下命令来安装 

#+BEGIN_SOURCE Bash
git clone https://github.com/zerodel/working_pipe_line.git
#+END_SOURCE
